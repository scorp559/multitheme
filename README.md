# MultiTheme - HTML Template Collection

A comprehensive, multi-purpose HTML template library featuring various pre-built website layouts and components for different industries and use cases.

## Overview

MultiTheme is an extensive collection of responsive HTML templates and components designed to accelerate web development. Whether you're building a corporate site, e-commerce platform, blog, portfolio, or specialized industry website, this template collection provides ready-to-use solutions.

## Getting Started

### 1. Download/Clone the Repository
```bash
git clone https://github.com/scorp559/multitheme.git
cd multitheme
```

### 2. View Templates
Simply open any HTML file in your browser:
- Start with `index.html` for the main demo
- Browse `index-*.html` files for industry-specific examples
- Check `components-*.html` for individual component showcases

### 3. Customize
Edit CSS and JavaScript files in the `css/` and `js/` directories to match your needs.

## Color Customization

Change the color scheme by:
1. Including a different theme file from `css/skins/`
2. Or editing `css/style.css` and `css/custom.css` directly

Example:
```html
<!-- Default theme -->
<link rel="stylesheet" href="css/style.css">
<!-- Change to blue theme -->
<link rel="stylesheet" href="css/skins/blue.css">
```
